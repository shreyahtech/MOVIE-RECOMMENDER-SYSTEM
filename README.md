# Movie Recommendation System

## Overview
A content-based movie recommendation system built using Python and Scikit-learn. It recommends movies based on similarity of genres, keywords, cast, crew, and overview.

## Features
- Search movies by title
- Content-based recommendations
- Cosine similarity
- NLP preprocessing

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK

## Dataset
TMDB 5000 Movies Dataset

## How to Run
1. Clone the repository
2. Install dependencies
3. Run the notebook or Streamlit app
